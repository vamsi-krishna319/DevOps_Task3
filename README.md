# Task 3 - Terraform + Docker

This task provisions a Docker container using Terraform.

## Steps Done
- Installed Terraform & Docker
- Created `main.tf` using Docker provider
- Ran `terraform init`
- Ran `terraform plan`
- Applied using `terraform apply`
- Verified container on port 8080
- Destroyed infra using `terraform destroy`
